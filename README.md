# mana-vietne
## Ceļošana pa Latviju
Ceļošana pa Latviju ir lielisks veids, kā iepazīt valsts dabu, kultūru un vēsturi. Latvijā ir daudz skaistu vietu, piemēram, jūras piekraste, meži, upes un vēsturiskas pilsētas. Ceļojot pa Latviju, var atpūsties, izbaudīt dabu un uzzināt ko jaunu.
## Sadaļas
- Sākumlapa
- Par tēmu
- Fotogalerija
- Kontakti
## JavaScript funkcijas
- Datu validācija formā – pārbauda, vai visi lauki ir aizpildīti
- Paziņojums par kļūdām – informē lietotāju, ja ir ievades kļūdas
